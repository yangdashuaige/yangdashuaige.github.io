python基础知识

发表于 2018-12-16 | 分类于 Python
  1. 输出：print()函数，可以一次打印一个，也可以接受多个字符，用逗号隔开，就可以成一串输出。如： print(‘100 + 200 =’, 100 + 200)。print()函数会依次打印每个字符串，遇到逗号就会输出一个空格。
  2. 输入。input()函数，当输入name=input()并且按下回车之后，Python交互命令行就等待输入，这时可以输入任意字符，然后按回车后完成输入。name=input(‘请输入你的名字’)
  3. python使用缩进来组织代码块，请务必遵守约定俗成的习惯，坚持用4个空格的缩进。

  4. 数据类型：

    1. 整型
    2. 浮点数
    3. 字符串：
      1. 是用单引号或者双引号括起来的任意文本，当字符串内既包含单引号也包含双引号的时候，使用转义字符\。
      2. 如果字符串中有很多的字符都需要转义，为了简化，使用r’’来表示内部的字符串默认不转义。
      3. 如果字符串内部有很多换行，用\n写在一行里不好阅读，为了简化，Python允许用’’’…’’’的格式表示多行内容
    4. 布尔值：True,False。只有这两种值。布尔值可以使用and,or 和not来计算。
    5. 空值：是一个特殊的值，用None来表示。None不能理解为0，因为0是有意义的，而None是一个特殊的空值。
    6. 变量
    7. 除法：
      1. 一种是/：计算结果是浮点数，即使两个整数恰好整除，结果也是浮点数
      2. //称为地板除，两个整数的除法仍然是整数
      3. %除余。

  5. 字符编码。因为计算机只能处理数字，如果要处理文本，就必须先把文本转换成数字才能处理。最早的计算机在设计时采用8个比特作为一个字节，所以，一个字节能表示的最大的整数就是255。Unicode将所有的语言都统一到一套编码里面，这样就不会再有乱码问题了。Unicode最常用两个字节表示一个字符。将Unicode编码转化成可变长编码的UTF8编码。UTF8编码把一个Unicode字符根据不同的数字大小编码成1-6个字节，常用的英文字母被编码成1个字节，汉字通常是3个字节，只有恨生僻的字符才会编码成4-6个字节。

  6. Python的字符串。在最新的Python版本中，字符串是以Unicode编码的，也就是说支持多语言。Python提供ord()函数获取字符的整数表示，chr()函数把编码转换为对应的字符。

    1. 由于Python的字符串类型是str，在内存中以Unicode表示，一个字符对应若干个字节，如果要在网络上传输，或者保存在磁盘上，就需要把str变为以字节为单位的bytes。Python中对bytes类型的数据用带b前缀的单引号或双引号表示：x=b’ABC’
    2. 以Unicode表示的str通过encode()方法可以编码为指定的bytes,例如：’ABC’.encode(‘ascii’)
    3. 当从网络或者从磁盘上读取了字节流，那么读到的数据就是bytes。要把bytes变为str,就需要decode()方法。
    4. 计算str包含多少个字符，可以用len()函数。len(‘ABC’)
    5. 计算包含多少字节len(b’ABC’),len(‘中文’.encode(‘utf-8’))

  7. 格式化，使用%来实现。%s表示用字符串替换，%d表示用整数替换，%f表示浮点数，%x表示十六进制整数。有几个%?占位符，后面就跟几个变量或者值，顺序要对应。如：’Hello,%s’ %’world’;’Hi,%s,you have $%d’ %(‘Mike’,10000)。

list和tuple

  1. list:是Python内置的一种数据类型，是列表，表示是一种有序的集合，可以随时添加和删除其中的元素。
    1. 定义：classmates=[‘Michael’,’Bob’,’Tracy’]
    2. 获取list元素的个数。len(classmates)
    3. 根据索引访问list每一个位置的元素，索引从0开始。当超出范围后会报一个IndexError错误，所以数组不要越界。另外，可以使用-1作为索引，表示倒数第几个元素，同样也有数组越界的问题。
    4. list是一个可变的有序表，可以往list中追加元素到末尾。classmates.append(‘Apple’);也可以把元素插入到指定的位置，比如索引号为1的位置：classmates.insert(1,’Jack’)
    5. 删除末尾元素，用pop()方法。classmates.pop();删除指定位置元素，classmates.pop(i)，里面放索引的位置。
    6. 把某个元素替换成别的元素，可以直接赋值给对应索引的位置
    7. list中的元素的数据类型也可以不同，比如：L=[‘apple’,123,True],里面的元素也可以是另一个list
  2. tuple。是另外一种有序的列表，和list非常类似，但是tuple一旦初始化就不能修改。索引没有append(),insert()方法。因为tuple是不可变的，所以代码更加安全。当定义一个tuple的时候，tuple的元素就必须被确定下来。

条件判断和循环

  1. 条件判断
    1. if的判断条件还可以简写，只要x是非零数值，非空字符串，非空list等，就判断为True,否则就判断为False。
    2. 可以使用input()函数读取用户的输入，但是因为input()返回的数据类型是str，str不能直接和整数比较，必须要把str转换成整数，可以使用int()函数来转换。但是当发现一个字符串并不是合法的数字时就会报错，程序就退出了。
  1. 循环。为了让计算机能计算成千上万次的重复运算，我们就需要循环语句。Python有两种循环：
    1. 一种是for… in循环，依次把list或者tuple中的每个元素迭代出来。
    2. 一种是while循环，只要条件满足，就不断循环，条件不满足时推出循环。

dict和set

  1. Python内置了字典：dict的支持，dict全称dictionary，在其他语言中也称为map,使用键值对进行存储，具有极快的查找速度。

    1. 定义：d={‘Michael’:95,’Bob’:80,’Tracy’:23}
    2. 取值：d[‘Michael’]
    3. 放值：d[‘Apple’]=67。因为一个key对应一个value，所以多次放会覆盖前面的数值。

    4. 当key不存在的时候，dict会报错，所以需要先判断。

      1. ‘Apple’ in d //会返回True或者False
      2. 通过get方法，如果不存在就会返回None,或者自己指定的value:d.get(‘Bpple’);d.get(‘Bpple’,-1)

    5. 删除，使用pop()方法，对应的value也会从dict中删除。d.pop(‘Apple’)

  2. dict和list比较：

    1. 查找和插入的速度极快，不会随着key的增加而变慢
    2. 需要占用大量内存，内存浪费极多

  3. Set和dict类似，也是一组key的集合，但不存储value,由于key不能重复，所以，在set中，没有重复的key.

    1. 定义：s=set([1,2,3])。要创建一个set,需要提供一个list作为输入集合。
    2. 添加：s.add(5)
    3. 删除：s.remove(key)可以删除元素

函数

  1. 函数定义：def 函数名(参数)：，然后在缩进块中编写函数体，函数的返回值可以使用return 语句返回。
  1. 函数的使用。如果你已经把my_abs()的函数定义保存为abstest.py文件了，那么，可以在该文件的当前目录下启动Python解释器，用from abstest import my_abs来导入my_abs()函数，注意abstest是文件名（不含.py扩展名）
  2. 空函数。可以使用pass语句def nop(): pass
  3. 内置函数isinstance()可以检查数据类型。
  1. 返回多个值。

其实Python函数返回的仍然是单一值，是一个tuple。

  1. 小结：

    1. 定义函数时，需要确定函数名和参数的个数
    2. 如果有必要，可以先对参数的数据类型做检查
    3. 函数体内部可以用return随时返回函数结果
    4. 函数执行完毕也没有return语句时，自动return None
    5. 函数可以同时返回多个值，但其实就是一个tuple

  2. 可变参数。在Python中，还可以定义可变参数，就是传入的参数个数时可变的，可以是1个、2个到任意个，还可以时0个。在函数定义的时候就要将函数的参数改为可变参数。

如果已经有一个list或者 tuple，要调用一个可变参数，可以在list或者tuple之前加一个*号，把list或者tuple变成可变参数传进去。

  1. 关键字参数。可变参数允许你传入0个或者任意个参数，这些可变参数在函数调用时自动组装成一个tuple。而关键字参数允许你传入0个或者任意个含参数名的参数，这些关键字参数在函数内部自动组装为一个dict。关键字参数可以扩展函数的功能，比如，在person函数里面，我们保证能够接收到name和age这两个参数，但是，如果调用者愿意提供更多的参数，我们也能够收到。试想你正在做一个用户注册的功能，除了用户名和年龄时必须填的，其他项都是可选项，利用关键字参数来定义这个函数就能满足注册的需求。
