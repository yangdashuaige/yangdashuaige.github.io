方法/函数

一种封装的思想。将特定的功能指令封装成一个模块。

好处如下：

  • 增加了代码的可读性
  • 提高了代码的反复利用率
  • 是封装的一种体现

函数和函数之间是同级的，所以函数的定义不能写在其他函数内部

函数的重载：方法名相同，但是参数的类型或者参数的个数不一样，返回值不考虑

数组

数组的定义：是一种复杂的数据类型 []，一组数据都是相同数据类型。除了基本数据类型之外的所有类型，都需要手动在堆里开辟空间。通过下标来取数据元素

栈里存储的变量一定要初始化。而在对于在堆里面的东西，java虚拟机会自动为其赋值

  • 整型 0
  • 浮点型 0.0
  • char ‘\u0000’
  • boolean false
  • 其他 null

数组一旦定义了长度，那么长度不可改变
