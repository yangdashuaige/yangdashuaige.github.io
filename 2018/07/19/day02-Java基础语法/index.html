提问：

  1. 垃圾回收机制？
    相比于c 和C++，这是java的一个优势。它使得程序员在编写程序的时候不用考虑内存管理。垃圾回收可以有效的防止内存泄露，有效的使用空闲的内存。任何一种垃圾回收算法都需要做两件事：
    • 发现无用的信息对象
    • 回收无用对象占用的内存空间

标识符

用来给类、对象、方法、变量、接口和自定义数据类型命名的。在Java中，凡是自己起名字的地方，都叫标识符，都遵循标识符规则。

命名规则：

  1. 由数字，字母，下划线，美元符号，人民币符号
  2. 以字母，下划线，美元符号开头
  3. java标识符对于字母大小写敏感，长度没有限制。标识符不能使用关键字
  4. 驼峰式

关键字

Java提供的的一些特殊含义的字母，全部小写

常量

不会改变的变量、直接定义的数据、被final修饰的

变量

程序运行过程中会变的量。存储的空间不变，所以对于计算机而言，只需记住变量的地址。空间的开辟在栈内。变量的内涵：一块内存地址。究竟内存空间开辟的大小，根据数据的类型来确定。

类型

* 整型：byte 1字节,short 2字节,int 4字节,long 8字节;
* 字符： char Unicode 2字节，可以存中文
* 浮点型：float 4，double 8；
* Boolean类型：boolean 没有规定，默认一个字节
* 其他数据类型：复杂的。
* 对于八个基本数据类型，输出变量名就是输出变量内存里面存储的值

字节：计算机存储资源的单位。
01的模式，1位，8位一个字节

byte范围

  • 原码（与10进制转换）
  • 反码
  • 补码，在计算机中以补码形式存储

对于正数来说，三码一致，对于负数：原码，除符号位取反得反码，反码+1=补码

数据类型转换

  • 自动转：小-》大 byte->short->int->long
  • 强制类型转换：会损失精度，会出错
  • boolean类型不能转换为其他的数据类型

所有基本数据类型都是在栈中开辟内存空间的。方法结束销毁。

运算符

  • 算术运算符 + - * / % ++ –
  • 关系运算符
  • 逻辑运算符： ！(取反)； &（与，全真为真，否则为假）； ^（异或，两者不同为TRUE）；|（或，只要有一个真就为真）； && （短路与，和&判断规则一样，只要有一个FALSE出现，后面就不运算了） ； ||（短路或，和|判断规则一样，只要有一个TRUE出现，后面就不运算了）
  • 位运算符：&(两个同为1才为1)、|（有1个为1就是1）、~（取反）、^（异或，只要不一样就是1，否则为0）、<<、>>、>>>
  • 赋值运算符：=、+=、*=、/=、%= 在运算结束以后会自动进行一个强制类型转换

所有的算术运算：byte ，short，char都先转成int类型再运算。
