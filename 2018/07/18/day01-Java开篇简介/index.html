day01-Java开篇简介

发表于 2018-07-18 | 分类于 洗心革面学Java

第一天

电脑

计算机（Computer）全称：电子计算机，俗称电脑。是一种能够按照程序运行，自动、高速处理海量数据的现代化智能电子设备。由硬件和软件所组成，没有安装任何软件的计算机称为裸机。常见的形式有台式计算机、笔记本计算机、大型计算机等。
状态：1和 0

程序

  • 指令：指导计算机做的事情，一系列指令成为程序
  • 操作系统程序： 合理分配调度硬件 windows mac Linux android
  • 应用程序：运行在操作系统之上的完成功能，软件（产品）

编程

编程语言

  1. 低级语言：尽可能让机器能够理解机器语言，不利于开发者
  2. 汇编语言：符号式，简单的加一些英文单词，汇编的机器将符号语言编译成二进制机器识别码
  3. 高级语言：C C++ Java Python PhP
  4. 编译语言：C C++一次性编译。缺点：不兼容
  5. 解释型语言：一边编译一边执行 js python，跨平台
  6. Java既属于编译型也属于解释型。一次性编译》字节码》在各大机器上解释执行（java虚拟机）

    思考：源码在Windows编译成二进制，放在Linux下可执行吗？

    不可以，平台不兼容。
    C C++属于一次性编译，不同平台不兼容即跨平台不兼容。它们属于编译语言。

环境

任何机器需要解释执行java编译好的字节码，必须安装java虚拟机。

java虚拟机跨平台吗？

安装软件（不跨平台）

jdk和jre区别联系 ？

JDK java开发包；JRE java运行环境

jdk目录小解

  • bin 可执行的指令 javac.exe编译，将java源文件转换为字节码文件； java.exe执行
  • include 头文件
  • jre java运行环境
  • lib 类库
  • db 数据库
  • src.zip 源码 java是开源的

在cmd中只有C:\Windows\System32文件夹中的指令可以识别，必须进入到指令所在的文件夹中执行指令

编译执行过程

  • javac xxx.java 编译java文件将.java文件变为字节码.class
  • java xxx 执行.class文件
  • Path变量的作用：让指令被Windows的DOS直接识别
  • CLASSPATH 变量的作用： 指明字节码文件所在位置

Java的优势

  • 可移植
  • 免费
  • 垃圾回收机制

IDE开发工具

  • Eclipse
  • Intellij Idea(WebStorm PhpStorm)

eclipse快捷键

  • alt+/ 手动提示
  • Ctrl+/ 注释（单行）
  • 选中然后 shift+ctrl+/ 多行注释
  • shift+ctrl+| 去除多行注释
  • syso 回车
  • main 回车
  • ctrl+alt+↑ 向上复制当前行
  • Ctrl+alt+↓ 下一行复制
  • Ctrl+d 删除
  • 修改快捷键 window》preference》General》Keys
