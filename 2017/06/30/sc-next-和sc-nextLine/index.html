sc.next()和sc.nextLine()

发表于 2017-06-30 | 分类于 Java

next()方法读取到有效字符以后才可以结束输入，对输入有效字符前遇到的空格键，Tab键或ENTER键等结束符，
next()方法会自动将其去掉 ，只有在输入有效字符后，next()方法才将其后输入的空格键，Tab键，Enter键视
为分隔符或者结束符。简单地说，next()查找并返回来自扫描器的下一个完整标记。完整标记的前后是与分隔
模式匹配的输入信息，所以next（）方法并不能得到带空格的字符串。
而nextLine()方法的结束符只是ENTER键，所以它是可以得到带空格的字符串的。而最重要的就是next()和nextLine()方法的连用
