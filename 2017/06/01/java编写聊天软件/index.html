java编写聊天软件（Java实验6）

发表于 2017-06-01 | 分类于 Java
  1. 具体要求：
    ① 创建项目及类：
    类名称：ChatClient
    项目：ChatRoomPrj
    包：默认包
    ② 组件构成：该JFrame中有六个组件：中间区为输出文本域JTextArea组件，南区为输入文本域JTextField组件，右边有两个JButton组件（Send和Quit），一个JComboBox组件用于选择发送消息的人名（昵称），以及一个菜单组件。
    ③ 使用嵌套面板和布局管理器来帮助构造上图所示的布局。具体如下所示：
    ※ 提示：JFrame默认采用BorderLayout，JMenuBar放置在North区，JTextArea放置在WEST区，JTextField放置在SOUTH区，CENTER区使用嵌套面板，该面板采用GridLayout（3行1列），分别添加两个JButton以及一个JComboBox组件。
    ④ 创建ActionListener，当按下Send按钮时，将JComboBox组件选择的人名，与JTextField输入文本域的文本一起复制到JTextArea输出文本域。
    ⑤ 创建ActionListener，在输入文本域按下 Enter键时，将ComboBox组件选择的人名（昵称），与JTextField输入文本域的文本复制到JTextArea输出文本域。
    ※ 提示：可以和上述 ④ 使用同一个监听器，以减少代码冗余。
    ⑥ 创建ActionListener，当按下Quit按钮时，程序退出。
    ※ 提示：监听器可以使用内部类实现，退出使用System.exit(0)。
    ⑦ 添加File菜单，该菜单必须包含Quit菜单项，选择该菜单项即终止程序。
    ※ 提示：和上述 ⑥ 使用同一个监听器，以减少代码冗余。
    ⑧ 添加Help菜单，该菜单必须包含About菜单项，可从该菜单项弹出一个消息对话框，显示有关程序和开发人员的备注信息。
