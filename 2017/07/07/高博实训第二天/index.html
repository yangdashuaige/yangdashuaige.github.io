高博实训第二天

发表于 2017-07-07 | 分类于 web前端

首先介绍元素的外观，包括三个部分：
selector border text
1.选择器一般有三种：类选择器，标签选择器和ID选择器。
它们的优先级是：ID>标签>类
2.介绍了基本的属性，如：height,width,border,background-color,margin,padding.
其中margin指的是外边距，它的值是会重合的，取较大的那个。padding一般用来美化，而不是用来定位。
margin:0 auto;表示自动居中。

基本上每个css上都会有这一段代码，sans-serif表示一类的字体，常用华文 方正。
font-size:40px; 字体大小
color:grey; 字体颜色
font-weight:900; 字体加粗
text-decoration:none;一般超链接下面有一个下划线，通过text-decoration来去掉下划线
letter-spacing:10px; 字母之间的间距
word-spacing:10px; 单词之间的间距
text-align:center; 规定元素中文本的水平对齐方式
line-height:90px; 设置行间的距离 行高
out-line:none; （轮廓）是绘制于元素周围的一条线，位于边框边缘的外围，可起到突出元素的作用。
text-shadow:3px 3px 5px blue; 分别为x y z轴的偏移量 最后一个是颜色
表示行内元素
其次主要讲了页面的布局
float:left; 将块级元素并列排版，但是会导致高度塌陷 这是加上overflow:hidden;
浮动以后想要换行用 clear:both;
容易写一些固定 的页面 如个人博客

在使用绝对定位后就可以用上下左右了
position:absolute;

position:static absolute relative fixed
absolute 的坐标远点为非static元素的左上角
relative 相对于自己在文档流中的位置 它可以用margin:auto

.div-relative{
width:100px;
height:100px;
background-color:yellow;
position:relative;
left:100px;
top:100px;
}

fixed:相对于浏览器 一般用作固定导航页面
