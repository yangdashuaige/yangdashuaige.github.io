Java作业四

发表于 2017-05-15 | 分类于 Java

  1. 求s=a+aa+aaa+aaaa+aa…a的值，其中a是一个数字。
    例如：2+22（此时共有2个2相加），3+33+333（此时共有3个3相加），
    5+55+555+5555+55555（此时共有5个5相加）
    请由键盘输入求和的项数a，给出计算结果。（10分）

  2. 按要求完成下列任务：（10分）
    （1）定义一个接口，包含一个有两个参数的area方法，用于计算面积；（2分）
    （2）定义描述圆的Circle类，数据成员为半径radius，并根据需要补充必要的方法；（3分）
    （3）定义描述矩形的Rectangle类，其数据成员为长度length和宽度width，并根据需要补充必要的方法；（3分）
    （4）定义一个测试类，通过输入一个圆的半径和一个矩形的长和宽，对所定义的类和接口进行测试。（2分）
