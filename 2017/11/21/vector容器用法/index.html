vector容器用法

发表于 2017-11-21 | 分类于 高博

vector类称作向量类，它实现了动态数组，用于元素数量变化的对象数组。和数组一样，vector类也用从0开始的下标表示元素的位置，但是当vector对象创建后，数组的个数会随着vector对象元素的个数的增大和缩小而自动变化。
1.创建
vector vecInt;
vector vecFloat;
vector vecA;
vector vecPointA;//自定义类型，保存指向类A的指针vector
