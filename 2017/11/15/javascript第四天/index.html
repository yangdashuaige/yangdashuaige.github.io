javascript第四天

发表于 2017-11-15 | 分类于 Java

一、Window对象
所有浏览器都支持window对象。他表示浏览器窗口。所有javascript全局对象、函数及变量均自动成为window对象
全局变量是window对象属性，全局函数是window对象方法
二、window的尺寸
window.innerHeight
window.innerWidth
浏览器窗口的内部高度和宽度
三、常用的方法
window.open() //打开新窗口
window.close() //关闭当前窗口
window.moveTo() //移动至当前窗口
window.resizeTo() //调整当前窗口的尺寸
四、window.screen
对象包含有关用户屏幕的信息

五、window.location对象用于获得当前页面的地址url,并把浏览器重定向到新的页面。
location.hostname 返回web主机域名
location.pathname 返回当前页面的路径和文件名
location.port 返回web主机的端口
location.protocol 返回使用的web协议
location.href返回当前页面的url

location.assign()方法加载新的文档

六、Window History
history.back() 与在浏览器点击后退按钮相同
history.forward() 与在浏览器中点击前进按钮相同

window.navigator对象包含有关访问者浏览器的信息

七、javascript消息框
有三种消息框 警告框 确认框 提示框

八、javascript计时
通过js可以在设定一个时间间隔之后来执行代码而不是立即执行。称之为计时事件。

var t=setTimeout(“javascript语句”,毫秒);
比如说一个简单的计时事件，点击按钮后，会在5秒后显示警告框

无限计时

cookie是存储于访问者计算机中的变量。每当同一台计算机通过浏览器请求某个页面时，就会发送这个cookie。可以使用js来创建和获取这个cookie的值。
