grtCurrentPosition()方法

获取地理位置信息 即经纬度

  1. successCallback():当成功获取地理信息使用回调函数的句柄
    successCallback()有一个参数Position对象
    1.1 coords:包含地理位置信息
    1.2 timestamp获取地理位置信息的时间

  2. errorCallback:当获取地理信息失败时使用的回调函数的句柄
    errorCallback有一个参数：positionError code:错误编号
    message:错误描述

  3. options：positionOptions
    获取用户位置信息的配置函数，有三个可选的属性
    enableHighAccuracy
    timeout
    maximumAge
    数据格式： JSON

例子：使用getCurrentPosition()方法获取地理位置信息

watchPosition()方法

监听和跟踪用户的地理位置信息
var warchId=geolocation.watchPosition(successCallback,errorCallback,options);
与getCurrentPosition()区别
持续告诉用户位置的改变
尤其是用户移动的时候，更有利于跟踪用户的位置

clearWatch()

停止监听和跟踪用户的地理位置信息
geolocation.clearWatch(watchId);
