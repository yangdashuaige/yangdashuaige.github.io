jQuery的遍历

  1. 遍历祖先
    parent()
    parents()所有祖先，直到
    parentsUntil():返回介于给定两个元素之间的所有祖先元素

  2. 遍历后代
    children():直接子元素
    find(): 返回被选元素的后代元素，一路向下直到最后一个后代

  3. 遍历同胞
    siblings() 所有同胞
    next() 下一个同胞元素
    nextAll() 所有跟随的同胞
    nextUntil()
    pre()
    preAll()
    preUntil()

基本过滤选择器

三个最基本的过滤方法是 first() last() eq(),他们允许宁基于其在它们允许您基于其在一组元素中的位置来选择一个特定的元素。
其他过滤方法，比如 filter() 和 not() 允许您选取匹配或不匹配某项指定标准的元素。

eq() 方法返回被选元素中带有指定索引号的元素。
索引号从 0 开始，因此首个元素的索引号是 0 而不是 1
