    属性可以建立列式布局，布局的一部分和另一部分重叠，还可以完成以前需要通过多个表格来实现的效果

定位：允许你定义元素相对其正常位置应该出现的位置，或者相对

一切皆为框
块级元素：div p h1
行内元素： span a strong
显示在一行里面，行内框

display: none /block
display 属性可以改变生成框的类型
block：让行内元素表现的和块级元素一样
none: 让生成的元素根本就没有框，这样该框和所有内容就不在显示，不会占用HTML文档的空间

CSS中有三种定位机制：
普通流： 除非专门指定，否则所有框都在普通流中定位，即元素的位置有HTML文档中的位置决定
浮动：float
left
right
none
inherit
浮动的框可以向左或者向右移动，直到它的外边缘遇到包含框或者另一个浮动框为止

绝对定位

position属性：
static:元素框正常生成，块级元素生成一个矩形框，作为文档流的一部分；行内元素创建一个或者多个行框，置于其父元素中
relative： 相对定位
元素框偏移某个距离，但是元素保留未定位前的形状，所占空间也保留

absolute:绝对定位， 元素框从文档流中完全删除，并相对于去包含快定位，包含快可能是文档中的另一个元素或者初始包含快。
元素在原先文档流中所占空间会关闭，就好像元素从来就不存在一样。元素定位以后会生成一个块级元素，而不论原来他在正常流生成的任何类型
fixed：类似absolute,其包含快是视窗本身
