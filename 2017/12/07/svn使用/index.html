svn使用

发表于 2017-12-07 | 分类于 svn工具

svn的使用 （默认已经安装过了）

  1. 在自己本地新建一个文件夹，例如SAE。然后右击鼠标，出现 SVN
    Checkout，点击它。将SAE的SVN仓库地址复制到上面，然后OK。有一个Revision的单选框，可以选择迁出的版本，head revision是最近的版本。
  2. 修改。
    一旦我们队其中的文件进行修改，那么文件或者文件夹的图片就会发生 变化。正常状态下是一个绿色的勾号，如果变成红色说明已经被修改过了。然后右击修改后的文件，点击SVN commit，接着会弹出一个窗体 commit to显示将要发送的仓库的地址，recent messages 是输入最近的日志列表，在下面的框中写日志，记录修改的信息。然后点击OK，修改的信息就完成了。
  3. 添加。
    如果我们要添加新文件到SVN服务器，只需在文件夹下的空白处右击 SVN Commit，也可以在要添加的文件上右击 点击TortoiseSVN–>Add，然后弹出一个窗体，点击OK。这时，这样并没有将这个文件提交到SVN服务器，只是将文件比饿哦为源代码库的文件，并将其状态置为修改状态。之后要再SVNcommit一次，这个文件才真正被提交到SVN服务器上的代码库中。
  4. 更新
    在需要更新的文件夹上右击，点击SVNUpdate即可。
  5. 重命名
    在需要重命名的文件夹上右击tortoiseSVN->Rename，然后修改名字，点击OK，然后SVNcommit。
